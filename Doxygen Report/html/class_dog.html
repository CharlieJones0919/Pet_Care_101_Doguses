Detailed Description

This class contains the implementations of functions (or behaviour actions) that can be employed by the dog's FSM states, and the individual dog's personality and care state values.

Member Function Documentation

◆ EndItemUse()

Dog.EndItemUse ( )
inline

Calls the EndItemUse() function in Controller so if the Item was temporary its position can be made placable again. That function will then set the ItemInstance's user back to null and deactivate the instance if it was single use. If the dog was holding the Item it was using, the instance object is unparented from the snout and its Y-position reset back to what it was before being picked up. (This Y-position is set in PickUpTarget() before the instance is "picked up"). The fact "USING_ITEM" is set back to false and the dog's target Item and ItemInstance object are reset back to their default values.
