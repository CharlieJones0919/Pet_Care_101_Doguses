Detailed Description

Stores all the dog's "CareProperties" as defined in DogGeneration (e.g. Hunger, Rest, etc...), the care value's associated "states" (e.g. Exhausted, Tired, Rested, etc...), and its current actual numerical value. Also contains a function for updating the care value and sets the care property's subsequent current state from that new value based on the state's lower and upper qualification bounds.
