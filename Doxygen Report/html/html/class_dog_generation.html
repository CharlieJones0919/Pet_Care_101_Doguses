Detailed Description

This class is responcible for creating the Dog instances when triggered to by GameTime, and modifying the basic Dog prefab with additional breed specific physical attributes as defined in BreedData.txt. Also initialises dogs' other base data attributes like their Care/Personality values + their states/bounds, along with their ages and initial personality values.
